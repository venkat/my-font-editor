# Test Letter

Test editing a specific letter in the font editor.

## Arguments

- `letter` (optional): The letter to test (default: "A")

## Steps

1. Open the live site https://venkat.io/my-font-editor/ with puppeteer
2. Click on the "Edit letter..." button to open the character picker
3. Click on the specified letter in the character grid
4. Take a screenshot showing the letter's strokes on the canvas
5. Report what strokes are currently defined for this letter
